# 页面未找到

你访问的页面可能已经迁移。

如果你是通过旧链接访问（例如链接里包含 `.md`），系统会自动尝试跳转到新地址。

<script>
(function () {
  var path = window.location.pathname || "";

  // 1) 兼容旧式 markdown 链接：/a/b/index.md -> /a/b/
  if (path.endsWith('/index.md')) {
    var target = path.replace(/\/index\.md$/, '/');
    window.location.replace(target + window.location.search + window.location.hash);
    return;
  }

  // 2) 兼容其他 markdown 链接：/a/b/page.md -> /a/b/page/
  if (path.endsWith('.md')) {
    var target2 = path.replace(/\.md$/, '/');
    window.location.replace(target2 + window.location.search + window.location.hash);
    return;
  }
})();
</script>
